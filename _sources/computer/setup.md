# 配置操作系统

这一节介绍如何安装和配置一些常见的操作系统，以满足日常科研工作的需求。
对于地震学科研初学者，通常更推荐使用 Linux 或 macOS；如果日常办公仍然依赖 Windows，
则推荐通过 WSL 获取 Linux 环境。

如果不确定应该阅读哪一篇，可以参考以下建议：

- Fedora：适合希望直接使用 Linux 作为主力科研系统的读者
- Ubuntu：适合初次接触 Linux、希望选择更常见发行版的读者
- macOS：适合使用 Mac 作为主力科研设备的读者
- WSL：适合主要使用 Windows、但又需要 Linux 命令行环境的读者
