本章主要介绍 Linux 系统的入门基础。熟悉和掌握本章内容后，基本就可以在 Linux/macOS 系统上
开展日常研究工作了。若想深入了解 Linux，可以参考一些较系统的资料进行学习。工作中若遇到问题，
可以随时查阅。推荐通过以下方式或资料进行查阅或学习 Linux：

1.  使用 Google 搜索
2.  Linux 101

    - 主页：https://101.lug.ustc.edu.cn/

    - 内容：系统地介绍 Linux 基础

3.  Bash 脚本教程

      - https://www.bookstack.cn/books/bash-tutorial：书栈网版本，内容似乎更全
      - https://wangdoc.com/bash/：网道网站，正在建设中

    - 作者：阮一峰
    - 内容：系统地介绍 Bash 脚本以及部分 Linux 基础
