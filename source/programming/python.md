## Python 与 Anaconda

Linux/macOS 都内置了 Python，足够日常使用，但强烈建议不要使用系统内置的 Python，
而建议安装 [Anaconda](https://www.anaconda.com/products/individual)，并使用
`conda` 管理 Python 环境。详情请参考 {doc}`Anaconda <software:anaconda/index>`。
