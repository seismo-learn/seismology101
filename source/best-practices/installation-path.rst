软件安装最佳实践
================

软件安装路径：

- ``/opt``
- ``~/opt``
- ``~/src``
- ``~/bin``
- ``~/software``

~/software
----------

保存软件的原始压缩包，相当于是做一个备份。

~/src
-----

对于需要编译源码的软件，将源码放在该目录的子目录下，比如 ``~/src/fk``\ 。

~/opt
-----

对于无需编译可以直接安装的软件，或者需要从源码编译且安装的软件，可安装
到 ``~/opt/`` 的子目录下，比如 ``~/opt/TauP`` 和 ``~/opt/GMT``\ 。

~/bin
-----

对于简单的小工具，比如单个源码文件或单个可执行文件，可以直接复制到 ``~/bin`` 目录，
如 ``distaz``

/opt
----

大多数闭源软件或商业软件都默认安装到该路径下。
